# Intensivão Java Spring - DS List

> Projeto do evento Intensivão Java Spring da devsuperior

## Tecnologias

- Java
- Spring Framework
- Rest
- H2 database
- PostgreSQL

### Bibliotecas adicionais / dependências

- Spring Data JPA
